# 🏥 Hospital Emergency Room Dashboard

A comprehensive and interactive **Hospital Emergency Room Analytics Dashboard** developed using **Power BI** to monitor patient flow, emergency department performance, waiting times, patient satisfaction, and referral trends. This dashboard helps healthcare administrators and hospital management make informed decisions by providing real-time operational insights.

---

# 📖 Short Description

The **Hospital ER Dashboard** analyzes emergency room patient data to evaluate hospital performance and identify opportunities for improving patient care. It provides detailed insights into patient demographics, admission trends, waiting times, referral departments, and satisfaction scores through interactive visualizations.

Designed for healthcare professionals, hospital administrators, and analysts, the dashboard enables data-driven decision-making for optimizing emergency room operations and improving patient experience.

---

# 🛠 Tech Stack

The dashboard was built using the following tools and technologies:

- 📊 **Power BI Desktop** – Dashboard development and visualization
- 🔄 **Power Query** – Data cleaning and transformation
- 🧠 **DAX (Data Analysis Expressions)** – KPI calculations and dynamic measures
- 🗂 **Data Modeling** – Building relationships and optimizing the data model

---

# 📂 Data Source

**Dataset:** Hospital Emergency Room Patient Data

The dataset contains **9,216 patient records** collected during hospital emergency room operations.

### Data includes:

- Patient ID
- Admission Date & Time
- Gender
- Age
- Race
- Department Referral
- Admission Status
- Patient Satisfaction Score
- Waiting Time
- Case Management Status

The dataset is used to evaluate hospital performance, patient demographics, operational efficiency, and service quality.

# ✨ Features & Highlights

## 📌 Business Problem

Emergency departments receive thousands of patients every month, making it challenging for hospital management to monitor operational efficiency.

Important questions include:

- How many patients visit the ER?
- What is the average waiting time?
- Which departments receive the highest referrals?
- What percentage of patients are admitted?
- How satisfied are patients with hospital services?
- Which age groups visit most frequently?
- Are there seasonal or monthly admission trends?

Without an interactive dashboard, answering these questions requires significant manual effort.

---

## 🎯 Goal of the Dashboard

The dashboard was designed to:

- Monitor Emergency Room performance
- Track patient admission trends
- Analyze waiting times
- Measure patient satisfaction
- Understand demographic distribution
- Identify heavily utilized departments
- Support operational planning and resource allocation
